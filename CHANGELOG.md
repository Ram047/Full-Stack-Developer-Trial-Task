# Changelog

All notable changes to this project are documented here. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and adheres to Semantic Versioning.

## [1.0.0] - 2026-07-13

### Added

-   **Authentication & Security**: Database-backed session cookie validation with httpOnly/SameSite settings, session rotation, Zod schemas, and token bucket rate limits.
-   **Dashboard & Analytics**: KPI summary panels and high-performance pure SVG weighted forecasting and donut charts.
-   **Deals Pipeline Workspace**: HTML5 interactive drag-and-drop Kanban Board with optimistic updates, saving states, rollback protections, and undo triggers.
-   **Structured Search & Exports**: Keyset cursors pagination, debounced SQL searches, bulk select operations, streamable chunked CSV downloads, and custom printable PDF formats.
-   **Technical Architecture**: Entity relationship models, dynamic OpenGraph edge banners, XML sitemaps, robots.txt directives, and native Node test runners.
